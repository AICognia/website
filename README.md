# Cognia AI - AI Receptionist & Voice Agent Platform

Enterprise AI voice solutions for businesses. Transform your customer experience with intelligent automation.

## Features

- **Modern & Responsive Design**: Optimized for all devices
- **Next.js 16**: Server-side rendering with React 19
- **Tailwind CSS**: Fast and customizable styling
- **Framer Motion**: Smooth animations and transitions
- **TypeScript**: Type-safe codebase
- **Clerk Authentication**: Secure user authentication
- **Multi-language Support**: English and Turkish

## Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Environment Variables

Copy `.env.local.example` to `.env.local` and fill in required values:
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- `CLERK_SECRET_KEY`
